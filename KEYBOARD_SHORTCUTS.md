# Total Commander Keyboard Shortcuts Reference

This document provides a comprehensive reference of Total Commander's keyboard shortcuts, organized by functionality. This will serve as the basis for KrakenEgg's keyboard navigation system.

## Function Keys (F1-F12)

### Basic Function Keys
| Key | Action | Description |
|-----|--------|-------------|
| F1 | Help | Opens detailed help documentation |
| F2 | Refresh | Reread source directory (refresh windows) |
| F3 | View | List/view files |
| F4 | Edit | Edit files in default editor |
| F5 | Copy | Copy files to opposite panel |
| F6 | Move/Rename | Move or rename files |
| F7 | Create Directory | Create new directory |
| F8 | Delete | Delete selected files/directories |
| F9 | Activate Menu | Activate menu above source window |
| F10 | Menu Control | Activate left menu or deactivate menu |
| F11 | Extended View | View files in extended mode |
| F12 | Brief View | View files in brief mode |

### Ctrl + Function Keys
| Key | Action | Description |
|-----|--------|-------------|
| Ctrl+F1 | Brief Display | Switch to "Brief" view mode (names only) |

| Ctrl+F4 | Sort by Extension | Sort files by file extension |
| Ctrl+F5 | Sort by Date | Sort files by modification date/time |
| Ctrl+F6 | Sort by Size | Sort files by file size |
| Ctrl+F7 | (Unused) | Available for custom assignment |
| Ctrl+F8 | Directory Tree | Display directory tree |
| Ctrl+F9 | (Unused) | Available for custom assignment |
| Ctrl+F10 | (Unused) | Available for custom assignment |
| Ctrl+F11 | Show Programs | Show only executable programs |
| Ctrl+F12 | Show Custom | Show user-defined file types |

## Customization Notes

### Available for Custom Assignment
Many key combinations are left unassigned in Total Commander to allow users to customize them:
- Most Ctrl+Shift+Letter combinations
- Many Alt+Letter combinations
- Function keys with multiple modifiers
- Number pad combinations

### Default Behavior Override
Users can remap almost any keyboard shortcut through:
1. Configuration → Options → Misc → Redefine hotkeys
2. Direct editing of wincmd.ini file
3. Plugin-specific keyboard assignments

This comprehensive keyboard reference serves as the foundation for implementing keyboard navigation in KrakenEgg, ensuring compatibility with Total Commander muscle memory while allowing for macOS-specific enhancements.
