# KrakenEgg UI Mockup - Quick Start Guide

## 🚀 Instant Setup (Zero Dependencies)

```bash
# 1. Navigate to the mockup
cd krakenegg-ui-mockup

# 2. Install dependencies (one time only)
npm install

# 3. Start the mockup
npm run dev

# 4. Open http://localhost:3000 in your browser
```

**That's it!** No Rust, no Tauri, no complex toolchain needed.

## ⌨️ Essential Keyboard Shortcuts to Test

### Core Navigation
- **Tab** - Switch between panels
- **Enter** - Enter directory or open file
- **Backspace** - Go to parent directory
- **Arrow Keys** - Navigate file list

### File Operations
- **F5** - Copy files (shows copy dialog)
- **F6** - Move files (shows move dialog)
- **F8** - Delete files (shows delete dialog)
- **F7** - Create new directory
- **Shift+F4** - Create new file

### View & Selection
- **Ctrl+A** - Select all files
- **Space** - Toggle file selection
- **Ctrl+1** - Brief view mode
- **Ctrl+2** - Detailed view mode
- **Ctrl+H** - Toggle hidden files

### Quick Access
- **Cmd+?** - Keyboard shortcuts help
- **Alt+F7** - Find files dialog
- **Cmd+,** - Settings dialog
